# 在线一对一授课平台 ── 教师桌面端
## 关联项目
- [服务端](https://github.com/Kimentanm/mf-edu-server)
- [后台管理前端](https://github.com/Kimentanm/mf-edu-admin)

## 功能列表
1. 一对一视频
2. 上传课件到画板，支持PPT、WORD、PDF、Video
3. 画板可涂写、擦除、撤销、标记，可直接同步到学生端
4. 画板画笔可定义线宽、颜色
5. 教师可授权学生拥有画笔功能
6. 录屏功能
7. 学生开小差（学生端应用不在桌面的最上面）提醒
